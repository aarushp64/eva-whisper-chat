# EVA Whisper Chat Frontend

This package contains the React/TypeScript frontend for EVA Whisper Chat.

## Development

- Install dependencies with `bun install`
- Start the development server with `bun run dev`

## Structure
- `src/` - Main React source code
- `public/` - Static assets

## Notes
- This package is managed with Bun as part of the monorepo.
- Update API endpoints to point to the backend service in `../backend`.
